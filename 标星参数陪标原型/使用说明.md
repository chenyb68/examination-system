# 标星参数陪标原型

依据《标星参数.docx》标星项制作的功能演示原型，用于陪标展示。

## 如何打开

用浏览器直接打开本目录下的 `index.html` 即可（建议 Chrome / Edge）。

## 已覆盖标星功能

1. 考试项目列表 CRUD
2. 考核项目创建（详情组件 + 审批流 + 电子签名）
3. 考核项目发放（自动/手动 + 待办）
4. 考核项目审核（多级 / 同级多人）
5. 进度督导（催办）
6. 材料归档（单下载 / 按课程 ZIP）
7. 考核方案管理（形成性/终结性/历史版本/关联任务）
8. 自动生成答题卡（OMR/OCR/混合 + 二维码）
9. OCR 手写体识别与自动评分
10. 试卷 / 课程 / 班级成绩分析
11. 权限控制（管理权限 + 课程权限 + 题库隔离）
12. 建库申请（密码锁定 / 课程库 / 综合库）
13. 试题动态标签
14. 试卷查重

## 说明

- 界面偏功能演示，未做视觉精修。
- 按钮操作为前端模拟，数据均为示意。
